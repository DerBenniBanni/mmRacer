A Game for gamedev.js jam 2026, Theme: "Machines"

Play it here: https://derbennibanni.itch.io/m13kro-machines

The game is a loveletter to old games like "Micro Machines" (hence the Theme connection) and one of the main goals of the game is that the ZIP-File has no more than 13 kb, in honor of the annual js13kgames gamejam!

Controls:

1 Player use ArrowKeys or WASD to steer the car

2 Players arrow keys for player one, WASD for playe two

The game has no assets, every graphic and sound is generated by javascript code. The only use of AI was to help me with some vector-math (collision of rotated rectangles)

The cars are created using a variant of "spritestacking" to fake a 3d-look
