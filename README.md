# m13kro Machines
A game for gamedev.js jam 2026, Theme: "**Machines**"

The game is a loveletter to old games like "Micro Machines", in respect and honor of the annual js13kgames.com gamejam I took the not official challenge to write the game in less than 13kB - with success, as the game uses 12904 of the 13312 bytes :-)

Play it here: https://derbennibanni.itch.io/m13kro-machines

## Controls:
Play with 1 Player: use ArrowKeys or WASD to steer the car
Play with 2 Players: arrow keys for player one, WASD for playe two

## Menu:
* Keys **1** & **2** to select solo or local multiplayermode
* **Left**/**Right** to select the type of the players car
* **Up** to select the cars color
* Key **T** to toggle the 4 tracks

## Features
* The game has no assets, every graphic and sound is generated by vanilla javascript code. 
* dynamic splitscreen for local multiplayer
* the cars are rendered using some sort of "spritestacking" to make them look 3d-ish
* did I mention the total size of 13kB ;-)
* the only use of AI was to help me with some vector-math (collision of rotated rectangles)

### Some things I could not fit in the 13 kB and/or the jam-time:
* SoundFX for the cars and the crashes
* different backgrounds, the wood-texture of the kitchen table has to be enough
* more vehicle-types
* mobile controls, gamepads
* a better menu-gui
